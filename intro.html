    • introducciion al css

    CSS, siglas en inglés de "Cascading Style Sheets", es un lenguaje de estilo utilizado para definir la presentación de una página web. En otras palabras, CSS permite a los diseñadores y desarrolladores controlar el aspecto visual de un sitio web. Con CSS, es posible cambiar el color y tamaño de las fuentes, ajustar los márgenes y espaciados, agregar efectos de animación, crear diseños de página personalizados y mucho más. Una de las principales ventajas de usar CSS es que separa el contenido de una página web de su diseño, lo que significa que los cambios en la apariencia del sitio se pueden hacer sin alterar el contenido en sí. En resumen, CSS es una herramienta esencial para cualquier persona interesada en el diseño web, ya que permite crear sitios web modernos y atractivos con facilidad y eficiencia.

    <h1>Iniciar Sesión</h1>
    <form action="stile.html">

      <input type="password" placeholder="Contraseña">

    </form>
  </div>
</body>
</html>
