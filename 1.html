¿Que es Html?

HTML es el lenguaje con el que se define el contenido de las páginas web. Básicamente se trata de un conjunto de etiquetas que sirven para definir el texto y otros elementos que compondrán una página web, como imágenes, listas, vídeos, etc.

A continuacion te muestro lo mas utilizado en Html:



  • <p>: define un párrafo.
  • <a>: define un enlace.
  • <img>: define una imagen.
  • <ul>: define una lista sin orden.
  • <ol>: define una lista ordenada.
  • <li>: define un elemento de lista.

Como podras averte dado cuenta cada etiqueta tiene su forma de abrirse y cerrarse ejemplo li se utilizan los signo < son para almacenar los atributos > y para cerrarlos es < / li >, siempre se debe de empesar por la equiteta html luego sige head que es el encabezao de nuestra pagina y title lo conforma, luego sige body que seria el cuerpo de la pagina y luego se deberac cerrar para una mejor comprencion te dejo un ejemplo pequeño.

Ejemplo de código HTML

				
					<!DOCTYPE html>
					<html>
					<head>
						<title>Mi sitio web</title>
						<link rel="stylesheet" href="estilos.css">
					</head>
					<body>
						<h1>Bienvenidos a mi sitio web</h1>
						<p>Este es un sitio web de ejemplo.</p>
					</body>
					</html>
