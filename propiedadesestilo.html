Propiedades de estilo en css

Una propiedad de estilo en CSS es una instrucción que se utiliza para definir cómo se debe mostrar un elemento HTML en una página web. Las propiedades de estilo en CSS se utilizan para cambiar aspectos visuales de los elementos HTML, como su color, tamaño, posición, fuente y muchas otras características.

tipos de propiedades

  • background-color: Esta propiedad se utiliza para definir el color de fondo de un elemento.

  • color: Esta propiedad se utiliza para definir el color del texto dentro de un elemento.

  • font-size: Esta propiedad se utiliza para definir el tamaño de la fuente de texto dentro de un elemento.

  • text-align: Esta propiedad se utiliza para definir la alineación del texto dentro de un elemento.

  • padding: Esta propiedad se utiliza para definir el espacio entre el borde de un elemento y su contenido.

  • margin: Esta propiedad se utiliza para definir el espacio entre un elemento y los elementos adyacentes.

  • border: Esta propiedad se utiliza para definir el borde de un elemento, incluyendo su ancho, estilo y color.

  • display: Esta propiedad se utiliza para definir cómo se muestra un elemento en la página, por ejemplo, como un bloque o como un elemento en línea.

  • position: Esta propiedad se utiliza para definir la posición de un elemento en la página, por ejemplo, si debe ser fijo o relativo.

Ejemplo

Ejemplo de código HTML 


                <!DOCTYPE html>
                <html>
                <head>
                    <title>Mi página con propiedades CSS</title>
                
                
                </head>
                <body>
                    <header>
                        <h1>Mi página con propiedades CSS</h1>
                        <nav>
                            <ul>
                                <li><a href="#">Inicio</a></li>
                                <li><a href="#">Acerca de</a></li>
                                <li><a href="#">Contacto</a></li>
                            </ul>
                        </nav>
                    </header>
                    <main>
                        <section>
                            <h2>Sección 1</h2>
                            <p>Este es el contenido de la sección 1.</p>
                        </section>
                        <section>
                            <h2>Sección 2</h2>
                            <p>Este es el contenido de la sección 2.</p>
                        </section>
                    </main>
                    <footer>
                        <p>Derechos reservados &copy; 2023</p>
                    </footer>
                </body>
                </html>

Ejemplo de código CSS de la pagina de ejemplo


    body {
        background-color: #f2f2f2; /*Fondo de color gris claro*/
        color: #333; /*Color de fuente negro*/
        font-family: Arial, sans-serif; /*Fuente principal Arial o una fuente sans-serif*/
        font-size: 16px; /*Tamaño de fuente de 16 píxeles*/
        margin: 0; /*Eliminar margen predeterminado del body*/
        padding: 0; /*Eliminar relleno predeterminado del body*/
    }
    
    header {
        background-color: #333; /*Fondo de color negro*/
        color: #fff; /*Color de fuente blanco*/
        padding: 10px; /*Añadir un relleno de 10 píxeles a la parte superior e inferior del encabezado*/
    }
    
    h1 {
        margin: 0; /*Eliminar margen predeterminado del encabezado h1*/
    }
    
    nav ul {
        margin: 0; /*Eliminar margen predeterminado de la lista de navegación*/
        padding: 0; /*Eliminar relleno predeterminado de la lista de navegación*/
        list-style: none; /*Eliminar el estilo de viñetas de la lista de navegación*/
    }
    
    nav ul li {
        display: inline-block; /*Mostrar los elementos de la lista de navegación en línea*/
        margin-right: 20px; /*Agregar un margen derecho de 20 píxeles entre los elementos de la lista de navegación*/
    }
    
    nav ul li:last-child {
        margin-right: 0; /*Eliminar el margen derecho del último elemento de la lista de navegación*/
    }
    
    nav ul li a {
        color: #fff; /*Color de enlace blanco*/
        text-decoration: none; /*Eliminar subrayado del enlace*/
    }
    
    main {
        max-width: 800px; /*Establecer el ancho máximo de la sección principal a 800 píxeles*/
        margin: 0 auto; /*Centrar horizontalmente la sección principal en su contenedor*/
        padding: 20px; /*Agregar un relleno de 20 píxeles a la sección principal*/
    }
    
    section {
        margin-bottom: 20px; /*Agregar un margen inferior de 20 píxeles a cada sección*/
        padding: 10px; /*Agregar un relleno de 10 píxeles a cada sección*/
        background-color: #fff; /*Fondo de color blanco para cada sección*/
        border: 1px solid #ccc; /*Agregar un borde de 1 píxel sólido de color gris claro alrededor de cada sección*/
        box-shadow: 2px 2px 2px rgba(0,0,0,0.1); /*Agregar una sombra suave a cada sección*/
    }
    
    h2 {
        margin-top: 0; /*Eliminar el margen superior predeterminado del encabezado h2*/
    }
    
    footer {
        background-color: #333; /*Fondo de color negro*/
        color: #fff; /*Color de fuente blanco*/
        padding: 10px; /*Agregar un relleno de 10 píxeles al pie de página*/
        text-align: center; /*Centrar el texto en el pie de página*/
    }
