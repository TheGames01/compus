¿Como agregar audios?

Para agregarlos unicamente utilizaremos la etiqueta audio, contlos, source

A continuacion te muestro lo mas utilizado en Html:



Ejemplo de código HTML

				
					<!DOCTYPE html>
					<html>
					<head>
						<title>Mi sitio web</title>
						<link rel="stylesheet" href="estilos.css">
					</head>
					<body>
						<audio controls><source src="c.mp3" type="audio/mp3">
						</audio>
					</body>
					</html>
