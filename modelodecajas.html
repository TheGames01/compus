Modelo de cajas

El modelo de caja en CSS es una forma de entender cómo se muestra y se organiza el contenido de una página web. Imagina que cada elemento HTML es como una caja rectangular que encierra su contenido, incluyendo cualquier espacio adicional de relleno, borde y margen. Es como si cada elemento tuviera su propio marco y el modelo de caja describe cómo se muestra este marco en la página.

tipos de propiedades

  • width y height: establecen la anchura y altura de la caja del elemento respectivamente.

  • padding: define el espacio de relleno dentro del borde de la caja.

  • border: establece el borde de la caja, que puede ser de diferentes estilos, anchuras y colores.

  • margin:define el espacio en blanco alrededor del borde de la caja, separando un elemento de otros elementos de la página.

Ejemplo

Ejemplo de código HTML 


                <!DOCTYPE html>
                <html>
                  <head>
                    <meta charset="UTF-8">
                    <title>Ejemplo de modelo de caja en CSS</title>
                  </head>
                  <body>
                    <div>
                      <p>Contenido del div</p>
                    </div>
                  </body>
                </html>

Ejemplo de código CSS de la pagina de ejemplo


    /* Estilo para el div */
      div {
        width: 300px;
        height: 200px;
        padding: 20px;
        border: 1px solid black;
        margin: 50px;
      }
