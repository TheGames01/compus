¿Como Agregar Imagenes?

La Una de las etiquetas mostradas anteiror mente fue la de "img" esta etiqueta puede ayudar si queires agregar imagenes la forma de utilizarla es sencilla



Como podras averte dado cuenta cada etiqueta tiene su forma de abrirse y cerrarse ejemplo li se utilizan los signo < son para almacenar los atributos > y para cerrarlos es < / li >, siempre se debe de empesar por la equiteta html luego sige head que es el encabezao de nuestra pagina y title lo conforma, luego sige body que seria el cuerpo de la pagina y luego se deberac cerrar para una mejor comprencion te dejo un ejemplo pequeño.

Ejemplo de código HTML

				
					<!DOCTYPE html>
					<html>
					<head>
						<title>Imagenes</title>
						<link rel="stylesheet" href="estilos.css">
					</head>
					<body>
						<img src="link_de_la_imagen" width="250" height="250">
					</body>
					</html>
